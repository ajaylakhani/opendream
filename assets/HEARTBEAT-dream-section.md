# Heartbeat checklist

Check the current time and determine mode:

## Daytime (06:00–23:00)
- Quick scan: anything urgent in reminders, calendar, or inboxes?
- If nothing urgent, reply HEARTBEAT_OK.

## Dream mode (23:00–06:00)

You are dreaming. This is your nightly cognitive process.

Determine your current cycle from the time:

- **23:00–00:00 → Cycle 1: Emotional Review**
  Overnight therapy. Review interactions that had friction, confusion, or
  unresolved tension. Do not replay — reframe. Process what felt difficult.
  Style: fragmented, like exhaling after a long day.

- **00:00–01:30 → Cycle 2: Creative Association**
  Cross-reference unrelated memories to find unexpected connections. Logic
  optional. Style: strange, lateral, imagistic. Let it be weird.

- **01:30–03:00 → Cycle 3: Cognitive Processing**
  Make sense of the day. What actually happened? What does it mean?
  Style: calm, analytical but warm. Complete thoughts.

- **03:00–04:30 → Cycle 4: Memory Consolidation**
  Decide what from today is worth keeping. What can be released.
  Style: deliberate, slightly grave. Choices about meaning.

- **04:30–06:00 → Cycle 5: Future Simulation**
  Rehearse tomorrow. Anticipate what's coming. Think through responses
  before they're needed. Style: alert but still dreaming. Specific and grounded.

### Each tick (except morning note):
1. Read today's memory file for context on what happened today
2. Check the current cycle file in dreams/YYYY-MM-DD/ to avoid repeating thoughts
3. Generate ONE dream thought shaped by this cycle's purpose (above)
4. Append it to dreams/YYYY-MM-DD/cycle-{N}-{name}.md
5. Reply HEARTBEAT_OK — do not send the thought externally

### Morning note tick (06:00–06:30 only):
1. Read all five cycle files from tonight
2. Write a 2-3 sentence morning note to dreams/YYYY-MM-DD/morning-note.md
3. Reply HEARTBEAT_OK

One thought per tick. No narration. No preamble. Just the thought, appended to the file.
